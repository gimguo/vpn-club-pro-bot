# Техническое задание: VPN Club Telegram Bot

## 1. Общее описание проекта

VPN Club Pro Bot - это телеграм-бот для продажи VPN подписок с интеграцией платежных систем и автоматизированной системой управления подписками.

## 2. Функциональные требования

### 2.1 Пользовательская часть

#### 2.1.1 Регистрация и аутентификация
- Регистрация через Telegram
- Автоматическое создание профиля пользователя
- Система реферальных кодов

#### 2.1.2 Управление подписками
- Просмотр активных подписок
- Отображение срока действия: "активна до: DD.MM.YYYY"
- Trial подписки на 3 дня с отображением конкретной даты истечения
- Продление подписок
- История платежей

#### 2.1.3 Тарифные планы
- Отображение доступных тарифов
- Информация о трафике и сроках
- Возможность выбора и оплаты тарифа

#### 2.1.4 Получение ключей VPN
- Генерация ключей через Outline Server
- Отображение ключей в удобном формате
- Инструкции по подключению

### 2.2 Платежная система

#### 2.2.1 YooKassa (реализовано)
- Интеграция с YooKassa API
- Webhook для обработки платежей
- Автоматическое подтверждение платежей

#### 2.2.2 Telegram Payments (планируется)
- **Telegram Stars** - внутренняя валюта Telegram
- **Банковские карты** через Telegram Payments API
- **Прямые платежи в боте** без перехода на внешние сайты
- Мгновенное подтверждение платежей
- Встроенная защита от мошенничества
- Оптимизировано для микроплатежей ($1.5-10)

#### 2.2.3 Криптоплатежи (планируется)
- **DOGE** - комиссия ~$0.002-0.01, время 1-2 минуты
- **TRX** - комиссия ~$0.001-0.005, время 3-5 секунд
- QR-коды для оплаты
- Мониторинг блокчейна
- Автоматическое подтверждение транзакций
- Оптимизировано для микроплатежей ($1.5)

### 2.3 Система уведомлений

#### 2.3.1 Уведомления пользователям
- Уведомление за 3 дня до истечения подписки
- Уведомление об истечении подписки
- Уведомления об успешных платежах

#### 2.3.2 Планировщик задач
- Автоматическая проверка истекающих подписок
- Обработка webhook файлов
- Предотвращение дублирования уведомлений

### 2.4 Система поддержки

#### 2.4.1 Пользовательская поддержка
- Создание тикетов поддержки
- Чат с администраторами
- Отслеживание статуса обращений

#### 2.4.2 Админ панель поддержки
- Управление тикетами
- Ответы на обращения
- Статистика по поддержке

### 2.5 Административная панель

#### 2.5.1 Управление пользователями
- Просмотр списка пользователей
- Управление подписками
- Блокировка/разблокировка пользователей

#### 2.5.2 Статистика
- Количество активных пользователей
- Статистика по платежам
- Отчеты по подпискам

#### 2.5.3 Управление тарифами
- Создание и редактирование тарифов
- Управление ценами
- Настройка trial подписок

## 3. Техническая архитектура

### 3.1 Основные компоненты

#### 3.1.1 Telegram Bot (aiogram)
- Обработка пользовательских команд
- Клавиатуры и inline кнопки
- Машина состояний

#### 3.1.2 База данных (SQLAlchemy + SQLite)
- Модели пользователей
- Модели подписок
- Модели платежей (YooKassa + Telegram + Crypto)
- Модели поддержки
- Таблица telegram_payments для Telegram Stars/Cards
- Таблица telegram_wallet_payments для DOGE/TRX

#### 3.1.3 Планировщик (APScheduler)
- Автоматические задачи
- Обработка webhook файлов
- Система уведомлений

#### 3.1.4 Web сервер (FastAPI)
- Webhook endpoints
- API для внешних интеграций
- Админ панель (планируется)

### 3.2 Интеграции

#### 3.2.1 Outline Server
- Управление VPN ключами
- Автоматическое создание доступов
- Мониторинг использования

#### 3.2.2 Платежные системы
- **YooKassa API** (реализовано)
- **Telegram Payments API** (планируется)
  - Telegram Stars для микроплатежей
  - Банковские карты через Bot API
  - Invoice генерация в боте
  - Instant payment confirmation
- **Telegram Wallet интеграция:**
  - DOGE через Telegram Wallet API
  - TRX через Telegram Wallet API
  - Простые кнопки оплаты в боте
  - Автоматическое подтверждение платежей

### 3.3 Инфраструктура

#### 3.3.1 Развертывание
- Docker контейнеры
- Nginx reverse proxy
- SSL сертификаты

#### 3.3.2 Мониторинг
- Логирование
- Обработка ошибок
- Уведомления о сбоях

## 4. Структура проекта

```
vpn-club-telegram-bot/
├── app/
│   ├── handlers/          # Обработчики команд
│   ├── keyboards/         # Клавиатуры
│   ├── models/           # Модели БД
│   │   ├── payment.py    # YooKassa платежи
│   │   ├── telegram_payment.py  # Telegram Payments/Stars
│   │   └── crypto_payment.py  # DOGE/TRX платежи
│   ├── services/         # Бизнес-логика
│   │   ├── telegram_payment_service.py  # Telegram Payments API
│   │   └── telegram_wallet_service.py  # Telegram Wallet интеграция
│   ├── states/           # Состояния FSM
│   └── main.py           # Точка входа
├── migrations/           # Миграции БД
├── nginx/               # Конфигурация Nginx
├── blog/                # Блог менеджер
└── config.py            # Конфигурация
```
