# 🚀 Telegram Payments Integration

## 📋 Обзор

Интеграция с Telegram Payments API позволяет пользователям оплачивать подписки прямо в боте без перехода на внешние сайты. Поддерживаются два способа оплаты:

- **⭐ Telegram Stars** - мгновенные микроплатежи
- **💳 Банковские карты** - классические платежи

## 🔧 Настройка

### 1. Получение токена платежного провайдера

1. Свяжитесь с @BotFather
2. Выберите ваш бот
3. Перейдите в Bot Settings → Payments
4. Выберите платежного провайдера
5. Получите токен провайдера

### 2. Переменные окружения

Добавьте в `.env` файл:

```bash
# Telegram Payments
TELEGRAM_PAYMENT_PROVIDER_TOKEN=1234567890:TEST:XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

### 3. Миграция базы данных

Выполните миграцию для создания таблицы telegram_payments:

```bash
sqlite3 database.db < migrations/003_telegram_payments.sql
```

## 🎯 Пользовательский опыт

### Поток оплаты:
1. Пользователь выбирает тариф
2. Выбирает способ оплаты (Stars/Card/YooKassa/Crypto)
3. Подтверждает оплату
4. Получает invoice прямо в боте
5. Оплачивает в интерфейсе Telegram
6. Автоматически получает VPN ключ

### Преимущества:
- **Без редиректов** - пользователь не покидает бот
- **Мгновенное зачисление** - automatic payment processing
- **Безопасность** - защита от Telegram
- **Простота** - минимум кликов

## 🔄 Обработка платежей

### Автоматическая обработка:
- Telegram отправляет `successful_payment` в бот
- Бот автоматически создает подписку
- Генерируется VPN ключ
- Планируется уведомление об истечении
- Пользователь получает все данные

### Статусы платежей:
- `pending` - платеж создан, ожидает оплаты
- `succeeded` - платеж успешно завершен
- `canceled` - платеж отменен

## 🛠️ Техническая реализация

### Новые компоненты:

1. **Модель**: `TelegramPayment` - хранение данных о платежах
2. **Сервис**: `TelegramPaymentService` - работа с Telegram Payments API
3. **Клавиатуры**: `PaymentKeyboard` - интерфейс выбора способа оплаты
4. **Обработчики**: обработка successful_payment и callback'ов

### Основные методы:

```python
# Создание платежа Stars
payment = await telegram_payment_service.create_stars_payment(
    user_id=user.id,
    amount=Decimal("1.50"),
    tariff_type="trial"
)

# Отправка invoice
await telegram_payment_service.send_stars_invoice(
    chat_id=chat_id,
    payment=payment
)

# Обработка успешного платежа
payment = await telegram_payment_service.process_successful_payment(
    successful_payment_data
)
```

## 🔧 Troubleshooting

### Частые проблемы:

1. **Invoice не отправляется**
   - Проверьте токен провайдера
   - Убедитесь что бот имеет права на отправку invoice

2. **successful_payment не обрабатывается**
   - Проверьте обработчик в main.py
   - Убедитесь что payload корректный

3. **Ошибки в логах**
   - Проверьте подключение к БД
   - Убедитесь что модели импортированы
